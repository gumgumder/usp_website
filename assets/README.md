# Assets

Statische Dateien der Website:

- `logos/`: Firmen-, Kunden- und Partnerlogos
- `images/`: Fotos, Illustrationen und Hintergrundbilder
- `icons/`: Favicons und sonstige Icons

Dateinamen sollten klein geschrieben werden und Bindestriche statt Leerzeichen
verwenden, zum Beispiel `kunde-musterfirma.svg`.
